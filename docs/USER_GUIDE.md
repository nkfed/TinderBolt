 # Керівництво користувача (User Guide)
 
 Це посібник для користувачів Telegram‑бота з інтеграцією ШІ‑агента. Бот доступний публічно:
 
 - Telegram: @Oldnick_helper_bot — відкрийте чат і натисніть Start.
 
 Документ призначено для технічних користувачів і містить: як почати, доступні команди, приклади діалогів, обмеження, FAQ та розділ про приватність/зберігання даних.
 
 
 ## Як почати
 1. Відкрийте посилання в Telegram: `https://t.me/Oldnick_helper_bot`.
 2. Натисніть кнопку «Start» або відправте команду `/start`.
 3. У головному меню оберіть потрібний режим або скористайтеся командами зі списку нижче.
 4. Дотримуйтеся підказок у чаті: бот надішле інструкції та, за потреби, кнопки.
 
 Поради:
 - Якщо «застрягли» в якомусь режимі — введіть `/start`, щоб повернутися в головне меню.
 - Для вводу довгих повідомлень краще уникати «сирих» підкреслень `_` (див. FAQ щодо форматування).
 
 
 ## Доступні команди
 Нижче — короткий опис команд і того, що вони роблять.
 
 - `/start` — показує головне меню, скидає активний режим. Надсилає привітальне повідомлення, інструкцію та кнопки.
 - `/gpt` — режим запитань до ChatGPT. Надішліть будь‑який текст — отримаєте відповідь ШІ.
 - `/date` — «Спілкування з зірками». Оберіть персонажа кнопкою; далі спілкуйтеся, як у звичайному чаті з «персоною».
 - `/message` — допомога з перепискою: надішліть кілька повідомлень (ваша історія), потім натискайте кнопки «Написати повідомлення» або «Запросити на побачення», щоб отримати варіанти від ШІ.
 - `/profile` — генерація Tinder‑профілю: відповідайте на запитання, бот згенерує опис/профіль.
 - `/opener` — генерація першого повідомлення (опенера): відповідайте на кілька питань про співрозмовника, бот підготує текст для старту бесіди.
 
 
 ## Приклади діалогів
 Нижче наведені типові сценарії використання. Текст користувача позначено як «Ви», відповіді бота — «Бот».
 
 ### 1) Режим GPT (/gpt)
 - Ви: `/gpt`
 - Бот: (інструкція + зображення)
 - Ви: «Поясни різницю між TCP і UDP простими словами.»
 - Бот: «TCP — з’єднання з гарантією доставки… UDP — без встановлення з’єднання… приклади застосування…»
 
 ### 2) «Спілкування з зірками» (/date)
 - Ви: `/date`
 - Бот: «Оберіть персонажа» (кнопки: Аріана Гранде, Марго Роббі, Зендея, Райан Гослінг, Том Харді)
 - Ви: (натискаєте «Зендея»)
 - Бот: (фото + повідомлення) «Ваша задача — запросити за п’ять повідомлень!»
 - Ви: «Привіт! Побачила твій фільм — ти неймовірна!»
 - Бот (від імені персонажа): «О, дякую! Які фільми тобі подобаються?»
 
 ### 3) Допомога з перепискою (/message)
 - Ви: `/message`
 - Бот: (пояснення + кнопки «Написати повідомлення», «Запросити на побачення»)
 - Ви: «Ми познайомилися на вечірці, переписуємось тиждень.»
 - Ви: «Вона любить собак і подорожі.»
 - Ви: (натискаєте «Написати повідомлення»)
 - Бот: «Думаю над варіантами…» → «Привіт! Бачив новий dog‑friendly маршрут…»
 
 ### 4) Генерація профілю (/profile)
 - Ви: `/profile`
 - Бот: починає коротке опитування («Ім’я?», «Вік?», «Хобі?» тощо), збирає відповіді
 - Бот: повертає стислий, читабельний профіль, який можна вставити у Tinder
 
 ### 5) Опенер для першого повідомлення (/opener)
 - Ви: `/opener`
 - Бот: «Як звати партнера?»
 - Ви: «Олена»
 - Бот: «Скільки років партнеру?»
 - Ви: «27»
 - Бот: «Оцініть зовнішність: 1–10 балів?»
 - Ви: «9»
 - Бот: «Ким працює?»
 - Ви: «Маркетолог»
 - Бот: «Мета знайомства?»
 - Ви: «Серйозні стосунки»
 - Бот: «ChatGPT генерує ваше повідомлення…» → повертає готовий опенер
 
 
 ## Діаграма (Mermaid)

 ```mermaid

 ```
 
 
 ## Обмеження
 - Довгі повідомлення: відповідь може бути скорочена моделлю (ліміт токенів). Намагайтесь формулювати запити стисло.
 - Форматування Markdown: «сирі» підкреслення `_` можуть «ламати» розмітку. За потреби використовуйте HTML‑режим (бот іноді сам підкаже).
 - Перемикання режимів: у кожен момент активний лише один режим. Для скидання — `/start`.
 - Зображення: бот надсилає лише вбудовані ілюстрації сценаріїв; обробка користувацьких зображень не підтримується.
 - Швидкість і квоти: затримки можливі через обмеження Telegram або постачальника ШІ.
 - Мова відповідей ШІ: залежить від вашого запиту; формулюйте українською для стабільних україномовних відповідей.
 
 
 ## Приватність і зберігання даних
 - Введені вами повідомлення передаються до сервісу ШІ (OpenAI Chat Completions) для генерації відповіді.
 - Історія діалогу зберігається тільки в пам’яті процесу бота (тимчасово) для поточного режиму/сеансу; постійне збереження на диск або в БД відсутнє.
 - Після перезапуску бота або скидання режиму (`/start`) контекст може бути втрачений.
 - Не надсилайте конфіденційні дані (паролі, токени, номери карток тощо).
 - Бот не виконує профілювання користувачів і не передає дані третім сторонам, окрім сервісу ШІ, необхідного для роботи функціоналу.
 - Технічні журнали (логи), якщо увімкнені, містять службові повідомлення й можуть включати ваші запити; їхній життєвий цикл і політика зберігання визначаються власником бота.
 
 
 ## FAQ
 - Як перезапустити або «скинути» бота?
   - Введіть `/start` — відкриється головне меню, активний режим буде скинуто.
 
 - Бот мовчить або «думає» занадто довго.
   - Зачекайте до 30–60 секунд. Якщо не допомогло — повторіть запит або спробуйте пізніше: можливі квоти/затримки мережі.
 
 - Відповідь виглядає обірваною.
   - Сформулюйте запит коротше або «Продовжуй» — бот згенерує продовження.
 
 - Ламається форматування з підкресленнями `_`.
   - Використовуйте подвійні бектики `` або HTML‑режим. Бот також може надіслати службове попередження, якщо виявить потенційно «битий» Markdown.
 
 - Чи зберігається моя історія довго?
   - Ні, контекст зберігається в пам’яті лише для активної сесії. Постійне зберігання не реалізоване.
 
 - Чи можна змінити персонажа в режимі `/date`?
   - Так, натисніть іншу кнопку персонажа або викличте `/date` знову, щоб обрати іншого.
 
 - Чи є вартість за використання?
   - Для користувача в Telegram — зазвичай ні. Вартість взаємодії з ШІ несе власник бота (залежно від тарифів постачальника ШІ).
 
 - Як отримати опенер, якщо не пам’ятаю всі дані про співрозмовника?
   - Вкажіть те, що знаєте. Модель адаптує текст; за потреби можна перегенерувати, уточнивши вхідні дані.
 
 - Чи можна отримати профіль англійською?
   - Так, сформулюйте запит або відповіді англійською — модель підлаштується.
 
 
 ## Поради з ефективного використання
 - Формулюйте конкретно: «Напиши 3 варіанти…» або «Стисло, 2–3 речення».
 - Давайте контекст: «Її інтереси — подорожі, собаки; стиль — дружній, без сленгу».
 - Перевіряйте факти: у режимі GPT модель може помилятися або «галюцинувати». Критично важливу інформацію верифікуйте самостійно.
